Папка для PDF-библиотеки приложения.

Структура по темам и подтемам:
- Создавайте папки вида `archive_pdfs/<тема>/<подтема>/файл.pdf`.
- Примеры:
  - `archive_pdfs/5класс/геометрия/урок_1.pdf`
  - `archive_pdfs/5класс/дроби/тренажер.pdf`
  - `archive_pdfs/контрольные/вариант1/контрольная_1.pdf`

Как это работает в приложении:
- В разделе `📚 Библиотека` есть два фильтра: `Тема` и `Подтема`.
- После выбора показывается список PDF и кнопки скачивания.
- Если PDF лежит прямо в `archive_pdfs` (без папок), он попадает в `Общее / Общее`.
- Если PDF лежит в `archive_pdfs/<тема>/` (без подтемы), он попадает в `<тема> / Общее`.

Для Streamlit Cloud:
- Файлы и папки архива должны быть закоммичены в репозиторий.
